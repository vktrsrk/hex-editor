# HEX Editor

A Windows desktop application for viewing and editing binary files.

## Features
- Open and display any binary file in hex format
- Synchronized hex and text views
- Edit bytes directly
- Dark theme interface
- Copy, Cut, Paste, Undo operations
- Print support

## Technologies
- C# / WinForms
- .NET Framework 4.7.2

## How to Run
1. Clone the repository
2. Open `hex-editor.slnx` in Visual Studio
3. Restore NuGet packages
4. Build and run (F5)
